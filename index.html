Week 8: Data Learning

Skills and concepts: Machine learning. Python (PyTorch, TensorFlow)

This week focusses on Machine Learning (ML) and the difference between “supervised” and “unsupervised” learning, and the use of labelled and unlabelled data.
Task: Attend one (or more) mornings of Talking Economics. After this construct a chart that would (a) support or (b) refute an argument that is made there.

This graph supports the argument "The pandemic widened gender inequalities"

made by Prof. Sarah Smith at the Talking Economics event in Bristol

One of Prof. Sarah Smith's key points was that women took on the majority of extra household

work due to the pandemic. This data supports that view, showing an average of 11.4% absolute

difference in increased workload for women compared to men.

Weeks 9 & 10: Data patterns

Skills and concepts: Machine learning. Python (PyTorch, TensorFlow)

These weeks focus on how to use moments of a distribution—the spread and range of data—on correlation, and on steps to establish causation. We discuss, with examples, ways to calculate and visualise the results of this in-depth analysis.
Task: Build a chart with data from two separate sources. Plot these data using correlation, regression or some other technique. Build and embed a chart that uses advanced statistical methods, testing causality or distribution.
